# Cervezame - React app

Esta tienda se creó con create-react-app con la intención de generar una aplicación dinámica y que trabaje en SPA. 


## Tools

- React V 17.0.2
- React-bootstrap V 2.1.0
- React-icons V 4.3.1
- React-router-dom V 5.2.0
- Firebase V 8.8.0
